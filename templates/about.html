About This Project

This Flask Lab Project demonstrates modern web development practices and DevOps workflows. It's designed as an educational project to showcase the integration of various technologies.

Technologies Used

  • Flask: Lightweight Python web framework
  • Docker: Containerization for consistent deployment
  • GitHub Actions: Automated CI/CD pipeline
  • Python: Backend programming language
  • HTML/CSS: Frontend presentation layer

Key Features

  • Clean and responsive web interface
  • Automated testing with pytest
  • Docker containerization
  • Continuous Integration/Continuous Deployment
  • Version control with Git

Project Goals

The main objective of this project is to provide a complete example of a modern web application workflow, from development to deployment, incorporating best practices in software engineering and DevOps.
